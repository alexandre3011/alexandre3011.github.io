Aqui você encotra informações opcionais para criação de distros.

Kernel e software de atualização

Informações opcionais recomendas

Kernel novo e atualizado é uma boa alternativa ao sistema porque vai ter maior compatibilidade com mais drivers de qualquer tipo de eletronicos. Sofware de atualização é importante para um sistema para gerenciar as atualizações de pacotes e deixar mais fácil de usar sem precisar de terminal para fazer as seguintes atualizações 

Base de sistema

Debian/Ubuntu 

Fazer sistemas baseado em Debian/Ubuntu são opções muito interessantes já que o arquivo .DEB é fácil de instalar e tem muitas atualizações dispóniveis.

Programas e compatibilidade de AMD64

Informações como toda outras opcionais, mas está tem que seguir mais arrisca mesmo sendo opcional.

Programas atualizados antes de salvar o sistema pronto, softwares atualizados podem trazer melhor funcionamente do sistema e também dos demais programas que depende um do outro. Compatibilidade, geralemte AMD64 bits funciona melhor em qualquer tipo de computador mesmo que sejá um intel (geralmente).
